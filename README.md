# CyberHub - Premium Gaming Lounge Website

A modern, fully-featured website for a cyber cafe/gaming lounge built with HTML, CSS, and JavaScript.

## Features

### 🎮 Pages
- **Home Page (index.html)** - Hero section with services, pricing, and contact information
- **Booking Page (booking.html)** - Complete booking system with real-time price calculation

### ✨ Key Features

#### Visual Design
- Modern cyberpunk/gaming aesthetic with neon colors
- Responsive design (works on desktop, tablet, mobile)
- Smooth animations and transitions
- Gradient effects and glowing elements
- Dark theme optimized for gaming audience

#### Functionality
- **Navigation Bar** - Sticky navbar with smooth scrolling
- **Hero Section** - Eye-catching introduction with animated background
- **Services Section** - 6 service cards showcasing facilities
- **Pricing Section** - 3 pricing tiers with feature comparison
- **Contact Section** - Contact information and message form
- **Booking System** - Complete booking form with:
  - Date and time selection
  - Duration selection
  - Gaming plan selection
  - Station preferences
  - Additional services (snacks, drinks, recording, tournaments)
  - Real-time price calculation
  - Booking confirmation
- **Footer** - Links, hours, and social media
- **Mobile Responsive** - Hamburger menu for mobile devices

## Customization

### To update business information:
1. Edit contact details in `index.html`
2. Update phone numbers and emails
3. Modify address in contact section
4. Update social media links in footer

### To modify prices:
1. Update pricing tiers in `index.html`
2. Adjust prices in `js/booking.js` (planPrices and servicesPrices objects)

### To change colors:
1. Edit CSS variables in `css/style.css`
2. Update `:root` section with new color values

### To add more features:
1. Modify HTML structure as needed
2. Add CSS styling in `style.css`
3. Add JavaScript functionality in `script.js` or `booking.js`
