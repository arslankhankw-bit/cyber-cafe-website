# CyberHub - Indian Cyber Cafe Services

A modern, fully-featured website for an Indian cyber cafe offering internet services, printing, photo prints, and document services. Built with HTML, CSS, and JavaScript.

## Features

### 🖥️ Pages
- **Home Page (index.html)** - Hero section with services, pricing, and contact information
- **Booking/Services Page (booking.html)** - Service pricing and calculation

### ✨ Key Features

#### Visual Design
- Modern, professional aesthetic with Indian-friendly colors
- Responsive design (works on desktop, tablet, mobile)
- Smooth animations and transitions
- Dark theme optimized for readability

#### Functionality
- **Navigation Bar** - Sticky navbar with smooth scrolling
- **Hero Section** - Eye-catching introduction
- **Services Section** - 6 service cards:
  - Internet/Gaming Services
  - Document Printing
  - Photo Printing
  - Scanning & Xerox
  - Document Digitization
  - Video Conferencing
- **Pricing Section** - Detailed pricing for all services
- **Contact Section** - Contact information and message form
- **Services Catalog** - Complete service listing with:
  - Internet hourly rates
  - Print pricing (per page/document)
  - Photo print sizes and costs
  - Xerox/Scanning fees
  - Document services
  - Real-time service calculation
- **Footer** - Links, hours, and social media
- **Mobile Responsive** - Hamburger menu for mobile devices

## Pricing Plans

### Internet Services
- **Hourly Rate**: ₹30/hour
- **Half Day (4 hours)**: ₹100
- **Full Day (8 hours)**: ₹180
- **Student Package**: ₹25/hour (with ID)

### Printing Services
- **B&W Printing**: ₹3 per page (A4), ₹5 per page (A3)
- **Color Printing**: ₹10 per page (A4), ₹15 per page (A3)
- **Spiral Binding**: ₹50-100 (per document)

### Photo Printing
- **4x6 inch**: ₹15 per print
- **5x7 inch**: ₹25 per print
- **8x10 inch**: ₹40 per print
- **Passport Size (2x2)**: ₹5 per print
- **Bulk Order (50+)**: 20% discount

### Scanning & Xerox
- **Xerox/Photocopy**: ₹2 per page (B&W), ₹5 per page (Color)
- **Scanning**: ₹3 per page (600 DPI), ₹5 per page (1200 DPI)
- **Document Lamination**: ₹10-20 per sheet

### Document Services
- **Certificate Printing**: ₹50-100 each
- **ID Card Printing**: ₹80-150 each
- **Typing Services**: ₹100-200 per hour
- **Passport Photo Editing**: ₹50 per set

## How to Use

1. **Clone or download** the repository
2. **Open index.html** in your web browser
3. **Navigate** using the menu bar
4. **Click "Our Services"** to view pricing
5. **Select services** and calculate cost

## Customization

### To update business information:
1. Edit contact details in `index.html`
2. Update phone numbers and emails
3. Modify address in contact section
4. Update social media links in footer

### To modify prices:
1. Update pricing tiers in `index.html`
2. Adjust prices in `js/services.js`

### To change colors:
1. Edit CSS variables in `css/style.css`
2. Update `:root` section with new color values
